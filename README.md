# Система определения объема заказа на основе нечеткой логики

Проектная работа по дисциплине "Системы искусственного интеллекта"

## Описание

Программа для определения оптимального объема заказа у поставщика на основе алгоритмов нечеткой логики.

## Установка

1. Установите Python 3.8 или выше
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Запуск основной программы:

```bash
python src/fuzzy_order_volume.py
```

Запуск тестовых примеров:

```bash
python src/test_examples.py
```

## Структура проекта

- `src/` - исходный код программы
- `report/` - отчет по проекту
- `requirements.txt` - зависимости проекта

## Входные параметры

- Текущий уровень запасов (0-1000 единиц)
- Ожидаемый спрос (0-500 единиц/месяц)
- Срок доставки (1-30 дней)
- Стоимость единицы товара (100-10000 рублей)

## Выходной результат

Рекомендуемый объем заказа в единицах (0-2000 единиц)
