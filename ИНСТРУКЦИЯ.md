# Инструкция по использованию

## Установка

1. Установите Python 3.8 или выше
2. Установите библиотеки:
```
pip install numpy scikit-fuzzy
```

## Запуск

Запустите программу:
```
python src/fuzzy_order_volume.py
```

Или тестовые примеры:
```
python src/test_examples.py
```

## Ввод данных

Программа принимает 4 параметра:
- Текущий уровень запасов (0-1000)
- Ожидаемый спрос (0-500)
- Срок доставки (1-30)
- Стоимость единицы (100-10000)

## Результат

Программа выдает рекомендуемый объем заказа в единицах.
