# Как собрать отчет в PDF

## Вариант 1: Использовать Pandoc (рекомендуется)

1. Установите Pandoc: https://pandoc.org/installing.html

2. Объедините все файлы отчета в один:

```bash
cd report
cat 01_analysis.md 02_problem_statement.md 03_technical_specification.md 04_idef_model.md 05_algorithm.md 06_program_code.md 07_test_examples.md 08_conclusions.md > full_report.md
```

3. Конвертируйте в PDF:

```bash
pandoc full_report.md -o отчет.pdf --pdf-engine=xelatex -V mainfont="Times New Roman" -V fontsize=12pt
```

## Вариант 2: Использовать онлайн-конвертер

1. Скопируйте содержимое всех файлов из папки `report/` в один документ
2. Используйте онлайн-конвертер Markdown в PDF:
   - https://www.markdowntopdf.com/
   - https://dillinger.io/ (экспорт в PDF)

## Вариант 3: Вручную в Word

1. Откройте файлы из папки `report/` в любом редакторе
2. Скопируйте содержимое в Microsoft Word
3. Сохраните как PDF

## Структура отчета

Отчет должен содержать следующие разделы:

1. Анализ предметной области
2. Постановка задачи
3. Техническое задание
4. Модель предметной области в нотации IDEF
5. Разработка алгоритма решения задачи
6. Текст программы с комментариями
7. Контрольные примеры работы программы (3 шт.)
8. Выводы по результатам разработки ПО
